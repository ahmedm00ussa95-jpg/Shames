# الشمس - Al Shams ERP
## نظام إدارة متكامل للمحلات التجارية

![Version](https://img.shields.io/badge/version-6.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Firebase](https://img.shields.io/badge/backend-Firebase-orange)

نظام ERP/POS متكامل باللغة العربية مبني على Firebase Realtime Database، مصمم للمحلات التجارية والمطاعم والمحلات التجزئة والجملة.

---

## 🚀 المميزات الرئيسية

### 💼 إدارة المبيعات
- نقطة بيع (POS) متكاملة مع دعم الباركود
- دفع متعدد (نقدي + بطاقة + تحويل + آجل)
- تعليق الفواتير واستئنافها
- اختصارات لوحة المفاتيح (F1-F9)
- فواتير يدوية واحترافية مع طباعة وتصدير Word/PDF/صورة

### 🏪 نقطة البيع
| الاختصار | الوظيفة |
|----------|---------|
| `F1` | تركيز حقل الباركود |
| `F2` | فتح نافذة الدفع المتعدد |
| `F3` | تعليق الفاتورة الحالية |
| `F4` | عرض الفواتير المعلقة |
| `F5` | مسح السلة |
| `F9` | فتح / إغلاق الوردية |

### 📦 إدارة المخزون
- تتبع كل حركة مخزون بالتفصيل
- جرد مخزني مع تسوية تلقائية
- تحويلات بين المخازن
- تنبيهات المخزون المنخفض والمنتهي
- سجل حركات كل منتج

### 🏢 نظام الفروع المتعددة
- فروع غير محدودة بقاعدة بيانات مستقلة لكل فرع
- تبديل بين الفروع بنقرة واحدة (للمدير العام)
- تقرير إجمالي لكل الفروع

### 👥 الأدوار والصلاحيات
| الدور | الصلاحيات |
|-------|-----------|
| 👑 مدير عام | كل شيء + كل الفروع |
| 🔑 مدير فرع | كل عمليات فرعه |
| 💳 كاشير | POS + فواتير + مرتجعات |
| 📊 محاسب | تقارير + مالية + مصروفات |
| 📦 مسؤول مخزن | منتجات + مخازن + مشتريات |

### ⏰ إدارة الورديات
- فتح وردية مع رصيد الصندوق
- إغلاق مع تسوية نقدية كاملة
- طباعة تقرير الوردية (A5)
- سجل كل الورديات السابقة

### 📊 التقارير والتحليلات
- مبيعات شهرية (آخر 12 شهر)
- أعلى 10 منتجات مبيعاً مع الأرباح
- أداء الكاشيرين
- توزيع طرق الدفع
- تصدير Excel للمبيعات، المنتجات، الديون، المرتجعات

## 🛠️ التثبيت والتشغيل

### المتطلبات
- حساب Firebase (مجاني)
- متصفح حديث (Chrome, Firefox, Edge, Safari)
- خادم محلي بسيط (Live Server, XAMPP, أو أي HTTP server)

### خطوات التثبيت

**1. إنشاء مشروع Firebase**
```
firebase.google.com → إنشاء مشروع جديد → Realtime Database → إنشاء قاعدة بيانات
```

**2. الحصول على Firebase Config**
```
Project Settings → Your apps → Web app → Firebase SDK snippet → Config
```

**3. تحديث Firebase Config في index.html**

ابحث في `index.html` عن:
```javascript
const firebaseConfig = {
  apiKey: "...",
  authDomain: "...",
  databaseURL: "...",
  ...
};
```
واستبدلها ببيانات مشروعك.

**4. تشغيل الخادم المحلي**
```bash
# مع VS Code Live Server: انقر زر "Go Live"
# أو مع Python:
python -m http.server 8000
# ثم افتح: http://localhost:8000
```

**5. تسجيل الدخول الافتراضي**
```
اسم المستخدم: admin
كلمة المرور: admin123
```

> ⚠️ **مهم:** غيّر كلمة المرور فور تسجيل الدخول لأول مرة

## 🔧 Firebase Security Rules

في Firebase Console → Realtime Database → Rules، ضع:

```json
{
  "rules": {
    "ctg": {
      ".read": "auth != null",
      ".write": "auth != null"
    }
  }
}
```
